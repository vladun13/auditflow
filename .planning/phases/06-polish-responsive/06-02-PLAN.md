    The rule to add:

    ```css
    /* Keyboard focus indicators — indigo ring matching design system primary */
    :focus-visible {
      outline: none;
      box-shadow: 0 0 0 2px white, 0 0 0 4px #4F46E5;
    }
    ```

    Use `box-shadow` (not Tailwind @apply with ring utilities) because this is a global CSS rule that must work on any element regardless of Tailwind class compilation.

    The double box-shadow creates: 2px white offset ring + 2px #4F46E5 indigo ring, visually equivalent to `ring-2 ring-[#4F46E5] ring-offset-2`.

    Do NOT modify any other rules in the file. Do NOT touch the @theme block, :root variables, or .dark variables.
